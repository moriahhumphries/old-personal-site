Projects

Open Weather Map

An application created during Codeup's Javascript and Jquery modules. This utilizes two APIs: Mapbox and OpenWeatherMap.
The user can search using the input box, and the map will adjust to display the new city and create a pin on the user's searched location. The weather will dynamically update for each search.
The user can also drag and drop the pin to update the weather for the pin's location.

Coffee Project

Created with a partner during a pair programming exercise in Codeup's Javascript module.
This application allows for the user to select their desired roast of coffee, and search by name within the roast category.
